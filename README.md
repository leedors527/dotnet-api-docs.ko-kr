# .NET Docs

이 저장소는 .NET을 위한 작성 중인 설명서가 담겨있습니다. 여기에 기여하시려면 [기여 가이드](https://github.com/dotnet/docs/blob/master/CONTRIBUTING.md)와 [이슈 목록](https://github.com/dotnet/docs/issues)을 확인해주세요. 

.NET 설명서를 개선하고 완성할 기여자 분들을 환영합니다. [Xamarin](http://developer.xamarin.com/api/root/classlib/), [Mono](http://docs.go-mono.com/?link=root%3a%2fclasslib) 그리고 [Unity](http://docs.unity3d.com/Manual/index.html)도 이 설명서를 사용할 것으로 예상합니다.

이 프로젝트는 저희 커뮤니티에서 예상되는 행동을 명확히 하기 위해 기여자 협약이 만든 행동 강령을 채택하였습니다.

더 많은 정보를 보시려면 [.NET Foundation Code of Conduct](https://dotnetfoundation.org/code-of-conduct)를 확인해주세요.
